# HW4：Diffusion Models

## 包含内容

- **实验代码**：
  - 主实验文件：`hw4.ipynb`
  - 训练脚本：`q1.py`, `q2.py`, `q3_a.py`等
- **模型实现**：`model/`目录下，`Unet` 和 `DIT` 模型的代码
- **训练日志**: `txt`文件
  - 训练过程指标
  - 损失函数变化
  - 模型权重保存信息
- **实验结果**：`results/`目录下
  - 生成图片集合
  - 训练过程中模型的的损失情况
  - 实验过程中采样效果不佳的图片

- **实验报告**
